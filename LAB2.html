Unit Outline

COS10026

Web Technology Project

Aims

This project-based unit focuses on developing your skills to build web applications using web technologies. You'll work individually and in teams to tackle real-world challenges, build proficiency in selecting tools, and apply security, ethics, accessibility, and professional communication principles.

Unit Learning Outcomes

  1. Identify core web technologies and their functions.
  2. Implement a project plan using effective project management strategies.
  3. Apply teamwork frameworks and communicate effectively using verbal and tech-based methods.
  4. Utilise database query techniques for CRUD operations.
  5. Develop functional web applications with ethics, project management, and security practices.
  6. Explain and implement basic web security principles.
  7. Assess ethical considerations and security practices including accessibility and privacy.

Content

  • Core Front-End and Back-End Development
  • Project-Based Learning and Version Control
  • Project Planning and Project Management
  • Web Security, Ethics, and Accessibility
  • Database Query Techniques
  • Professional Communication and Presentation
  • Teamwork Framework
